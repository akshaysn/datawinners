XLSForm is a tool that will allow you to design a Questionnaire with advanced features:

  • Repeats (Repeated set of questions)
  • Skip logic
  • Grouping questions
  • Cascading selects
  • Answer Constraints
  • Calculations
  • Required and optional questions
  • Hints (Instruction texts)
  • Default answers

Read more: How to create XLSForms in DataWinners

Sample Forms

Download sample XLSForms to see how to create advanced Questionnaire features in Excel.
  1. Repeated set of questions, Skip logic, Cascading selects, Multi Choice questions: household_details.xls
  2. Grouping questions, Answer constraint, Calculations: health_survey.xls
  3. Hints, Required and optional, Default answers: nutrition.xls
